# ImagePro (图片大师)

**Live Demo**: https://imagepro.fandx.vip

ImagePro is a modern, client-side image processing tool built with React and Vite. It provides essential image editing capabilities like cropping, resizing, and compression directly in your browser.

## Features

-   **Privacy First**: No images are uploaded to any server. Everything is processed locally.
-   **Smart Cropping**: Presets for common aspect ratios (19:6, 4:3, 16:10).
-   **Optimization**: JPEG compression and format conversion.
-   **Responsive Design**: Professional darker-themed UI.

## Tech Stack

-   React 18
-   TypeScript
-   Vite
-   CSS Modules

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Run dev server: `npm run dev`.
4.  Build for production: `npm run build`.
